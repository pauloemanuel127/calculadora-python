# Calculadora em Python 🐍

Este projeto é um exercício de lógica para praticar os fundamentos do Python. Eu já tinha feito uma versão bem simples dessa lógica anteriormente em **Bash**, mas agora o foco foi aplicar os recursos do Python para deixar o código mais robusto e organizado.

### O que o código faz:
- **Operações Básicas:** Soma, subtração, multiplicação e divisão.
- **Tratamento de Erros:** O programa não "quebra" se você digitar letras; ele usa `try/except` para avisar que o valor é inválido e pede os números novamente.
- **Matemática na Divisão:** O código trata divisões por zero de forma específica, diferenciando **Indefinido** de **Indeterminação** ($0/0$).
- **Uso Contínuo:** A calculadora roda em loop (`while`) e só fecha quando você digita o comando `exit`.

### Meu objetivo:
Estou investindo no aprendizado de Python e usei este projeto para fixar conceitos de controle de fluxo, loops e tratamento de exceções. O foco foi escrever um código limpo, comentado e que entenda as intenções do usuário.

### Como rodar:
1. Tenha o Python instalado.
2. Baixe o arquivo `calculadora.py`.
3. Rode no seu terminal: `python calculadora.py`.
